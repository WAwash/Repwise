# Rep-Wise

Sometimes it can be overwhelming to remember what to do each day at the gym, which is why I created Rep-Wise. Rep-Wise is a website meant to help people get started at the gym or keep track of what muscles to train. Short demonstrations are also included with every exercise. Lastly, I would really appreciate feedback to help me improve my website, a Google Form for feedback is included in the website. 
Enjoy!
